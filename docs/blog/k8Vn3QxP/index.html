【弊社村木がテレビ出演しました！】芸能人vs元マッキンゼー　ケース面接やってみた

皆さま、こんにちは！
このたび、弊社代表の村木がテレビに出演いたしました！
「元気ジャパンTV」に出演し、芸能人3人にフェルミ推定挑戦をしてもらいました。
収録の様子の動画が公開されましたので、お知らせいたします。
今回のテーマは「芸能人vs元マッキンゼー ケース面接をやってみた」です！

コンサルティング業界への転職を目指している方であれば、一度は耳にしたことがある「ケース面接」。
実際の面接では、どのような思考プロセスが求められ、どのような観点で評価されるかを、より分かりやすくイメージしていただける内容となっております。
ケース面接対策に取り組まれている方はもちろん、コンサル転職に興味をお持ちの方にもご覧いただける内容です。
ぜひご視聴ください！
