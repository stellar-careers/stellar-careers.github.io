【体験記】エンジニアからコンサルタントへの転職を実現されたT/Kさんにインタビュ― 毎日の練習がキャリアを変えた

ーーーステラキャリアズとの出会いはどのようなきっかけだったのですか？

　エンジニアとして新卒5年目を迎えるころ、知人の紹介でステラキャリアズの村木さんに出会いました。最初のカジュアル面談では、そもそも転職すべきかどうか、今後のキャリアパスについて悩んでいることを相談しました。なので、転職相談というよりはキャリア相談という形で私の転職活動はスタートしたんです。

ーーーその面談を通して、キャリアについて何か変化はありましたか？

　はい。企画や戦略にも携われるビジネス視点を持ったエンジニアを目指していたのですが、村木さんとの面談を通してコンサルタントというキャリアが初めて選択肢に上がりました。「本当にやりたいこと」と「それを実現するキャリア」を徹底的に相談できたこと、さらにコンサルについても基礎から丁寧に教えていただいたことで、コンサルが目標を実現できる有力な選択肢だと確信できました。特に「エンジニア経験を活かしたい」という私の意思を尊重し、技術知見を活かせる企業選びまでサポートしてもらえたのは大きかったです。

ーーー職務経歴書の作成はどのようにサポートしてもらったのですか？

　初めて経歴書を書いた私に対して、村木さんは何度もWeb面談でフィードバックや添削をしてくださいました。面談では、過去の自分の経験を深掘りして、「コンサルでアピールできる要素」として書類に落とし込む作業を繰り返しました。結果的に5回ほど添削していただいたと思います。表現や文体の細かい部分まで指導してもらい、一人では気づけなかったアピールポイントを最大限盛り込めたことで、自信を持って選考に臨める質の高い経歴書が完成しました。

ーーー面接対策についてはいかがでしたか？

　選考が始まってからは、村木さんが面接対策に驚くほど多くの時間を割いてくださいました。特に第一志望の面接前は、「何回でも面接練習しましょう」と言っていただき、ほぼ毎日1日1回のペースで模擬面接を実施しました。最初は緊張して通り一遍の回答しかできなかったのですが、繰り返し練習することで自分の本心と向き合いながら話す内容の精度を上げられました。そのおかげで本番では自分の意思を明確に伝えられ、内定につながったと実感しています。

ーーー最後に、ステラキャリアズでのサポートについて一言お願いします。

　ここまで一人の候補者に時間を割いてくれるエージェントはなかなかいないと思います。面談の時間をフルに活用できたからこそ、納得のいく転職ができました。村木さんには本当に感謝しています。転職は「自分と向き合う時間」でもあると実感しました。同じように迷っている方にも、この経験をぜひ伝えたいです。
