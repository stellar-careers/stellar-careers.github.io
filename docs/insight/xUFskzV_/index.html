【体験記】年収2倍を実現！30歳目前で選んだコンサル転職のリアル

ステラキャリアズ広報室より、弊社の支援で転職成功された方の体験記をご紹介します。
今回は、スタートアップから外資系トップ戦略ファームへの転職を成功された20代のS/Nさん。
起業を見据えキャリアを積んできたものの、30歳を前に「収入を得ながらテーマに挑戦する」方向にシフト。ステラキャリアズとの出会いをきっかけに、自分では辿り着けなかった多様なコンサルファームを知り、視野が一気に広がりました。結果は――第一志望を含む複数ファームから内定、年収は約2倍に。
村木の伴走により「目的から逆算した選択」ができたことで、納得度の高い転職を実現しました。

転職は一人で悩むより、信頼できる伴走者と進むことで可能性が広がります。
あなたもまずはステラキャリアズに相談してみませんか？
未経験からのキャリアチェンジを目指す方、ぜひご覧ください！

はじめに（出会いのきっかけ）


自分で起業すること、独立することがずっとゴールでした。スタートアップ2社で、データアナリストと営業職という二足のわらじを履きながら、「事業を作る」というテーマを軸に20代を走り抜けてきました。
しかし、30歳を目前に、これからは自分のやりたいことの方向性や将来への投資も見据え、しっかりと収入を得ながら自分のテーマに取り組むフェーズに移るべきだと考えるようになりました。そんなタイミングで、現職でお世話になった上司が、ステラキャリアズの村木さんと同じ会社で働いていた縁から、ご紹介いただいたのがきっかけでした。

自分では辿り着けなかった企業との出会い


村木さんと話す中でまず感じたのは、「世の中には本当に多種多様なコンサルファームが存在する」という事実でした。
自分ひとりで情報収集していたら、知名度や条件面だけで判断してしまいがちです。しかし村木さんは、私の将来のゴールや身につけるべき経験を丁寧に言語化し、それに基づいた企業を提案してくれました。
当初の想定とは異なる選択肢も多く、その一つひとつが新しい景色を見せてくれたことで、「こういう道もあるのか」という気づきとワクワク感が生まれました。

複数内定からの決断


転職活動の結果、イノベーション系のファームと、米系グローバルのストラテジー×デジタルのコンサルティングファームから内定をいただくことができました。
どちらも魅力的で、意思決定は簡単ではありませんでしたが、最終的には後者を選びました。理由は、グローバル規模での戦略立案とデジタル実装の両方に関わる経験が、将来の独立を見据えたときに最も価値が高いと感じたからです。

年収2倍という成果とキャリアの可能性


今回の転職で、年収は約2倍となりました。数字のインパクトは大きいですが、それ以上に得られたのは、自分の市場価値を客観的に再認識できたことです。
これまでの経験を別の文脈で活かせる場があるとわかったことで、今後のキャリアの可能性が大きく広がりました。何より、「自分のテーマに向けて動き出せる準備が整った」という実感があります。

村木さんのサポートが光った場面


村木さんの最大の強みは、市場と業界構造を深く理解しつつ、候補者の目的から逆算して提案できることだと感じます。
私の場合も、応募企業の選定から面接戦略、意思決定の場面まで、一貫して「目的に立ち返る」サポートをしていただきました。選考中のフィードバックも非常に具体的で、迷ったときには必ず方向を修正してくれる存在でした。
結果として、納得感の高い転職活動を終えられたのは、村木さんの伴走があったからこそだと思います。

最後に（これから転職活動をする人へのメッセージ）


転職は、ゴールや条件から逆算して動くことが大事だと今回痛感しました。自分ひとりでは知り得ない企業や選択肢を知ることで、可能性は一気に広がります。
また、目先の年収やブランドにとらわれず、「自分が何を成し遂げたいのか」「どんな経験を積みたいのか」という軸を持つことが、納得度の高い意思決定につながります。
ステラキャリアズのように、候補者の目的を真剣に理解し、伴走してくれるパートナーの存在は心強いものです。これから転職を考える方には、ぜひ信頼できる伴走者と共に、自分らしいキャリアを描いてほしいと思います。
