【体験記】コンサルtoコンサル転職：年収150万アップを実現

ステラキャリアズ広報室より、弊社の支援で転職成功された方の体験記をご紹介します。
今回は、SIerからコンサル業界への転職に挑戦された20代のS/Yさん。
現職での業務に限界を感じ始め、「考える仕事」への関心から転職を決意。知人の紹介でステラキャリアズを利用し、初回面談から「候補者目線」の丁寧なサポートを受けたことをきっかけに、手ごたえを感じると、面接対策では自己理解を深める支援が功を奏し、第一志望からの内定を獲得。
未経験からのキャリアチェンジを目指す方、ぜひご覧ください↓↓↓

ステラキャリアズとの出会い


入社４年目のタイミングでキャリアアップを目指し、外資系コンサルから内資系コンサルへの転職を考え始めました。
社内のプロモーションの鈍化や国外の業績が芳しくないこともあり、転職するかの判断と、他企業の情報収集も含めご相談をさせていただきました。
コンサル経験者というのもあり、業界への知識の深さと自分が漠然と持っていた課題感を即座に言語化してくださり、この方にご協力いただきたいと強く感じたため、お世話になることを決めました。

コンサル出身ならではの品質の高さ

今回、私は第一志望含め選考を進めた企業４社全てから内定をいただくことができました。面接で評価いただけた理由として、以下を考えています。
・業界内での企業比較が明確になっていた
・転職理由や現職でのアピールポイントなどが具体的であった
・フェルミ推定などの対策が十分にできていたこれら全て、村木さんに対策を手伝っていただきました。相談事項についてもクイックにレスポンスをいtだくことができ、スムーズに課題解決できたのを覚えています。前職で利用させていただいたエージェントさんとは明らかに質の違いを感じました。
内定先を選ぶ際にも、私の内面や嗜好を理解した上で判断材料として情報を下さるので大変助かりました。

サポート体制

不明点があった際に、Lineで即レスポンスいただけるので大変助かりました。
選考の対策においても、面接での注意点やフェルミ推定で聞かれるような傾向を共有してくださり、時間がない中で大変スムーズに準備を進めることができました。
面接練習においても明確なフィードバックをいただくことができ、これがなければ自分の内定はなかっただろうなと振り返っています。

最後に

時間がない中で自分にリズムに合わせ品質の高い対策を実施していただいたことに、非常に感謝しております。
新しい職場でもしっかりと結果を出し、努力を続けようと思います。
