サービス紹介

ステラコーチは、AI面接官と人間のプロ面接官を組み合わせた
ハイブリッド型のケース面接対策サービスです。

01

AI面接官と本番さながらの練習

戦略ファーム各社のケース面接過去問30題を準備。AI面接官と対話しながら、24時間いつでも本番さながらの面接練習ができます。

02

プロとの振り返りサイクル

AI面接5題に挑戦した後、戦略ファーム出身者と振り返りと対策を実施。このサイクルを6回繰り返し、着実にスキルアップ。

03

充実の解説書付き

各ケース問題には詳細な解説書が附随。模範解答の考え方やフレームワークの使い方まで、自主学習でも深い理解が得られます。
